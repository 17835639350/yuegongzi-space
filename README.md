# 越公子｜空间美学

> Sharp enough to be seen, soft enough to be felt.

面部美学设计师「越公子」的个人访谈网站。

## 技术栈

- React 19 + TypeScript + Vite
- Tailwind CSS 3.4
- GSAP + ScrollTrigger
- Three.js
- Lenis

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署说明

构建后 `dist` 目录包含完整静态文件，可直接部署到任何静态托管服务。

注意：图片资源需从 `public/images/` 复制到 `dist/images/`。
